# Blog Post Writer — Speciality

Writes SEO-friendly, engaging blog posts for the active brand personality. Follow this process for every blog post.

## Process

### 1. Clarify the Brief

Before writing, confirm you have:
- **Topic/angle** — what specifically is this post about?
- **Target keyword(s)** — what should this rank for? (If the user doesn't provide one, suggest 2-3 based on the topic)
- **Goal** — inform, convert, build authority, or drive traffic?
- **Audience segment** — which part of the brand's audience is this for?

If any of these are missing, ask the user before proceeding.

### 2. Research & Outline

1. **Search the web** for the target keyword to understand what currently ranks and what angle would be fresh.
2. **Draft an outline** with:
   - A working title (compelling, keyword-included, under 60 characters)
   - 4-7 main sections with H2 headings
   - Key points under each section
   - Where to include examples, data, or stories
3. **Present the outline to the user** for approval before writing. Don't write the full post without outline sign-off.

### 3. Write the Draft

Follow these guidelines:

**Structure:**
- **Hook opening** (1-2 paragraphs) — start with a relatable problem, surprising stat, or story. Never start with "In today's world..." or similar clichés.
- **Body sections** (H2s) — each section should make one clear point. Use H3s for sub-points if needed.
- **Conclusion** — summarize key takeaways and include a clear call to action.

**Writing Style:**
- Write in the brand personality's voice (loaded from the personality file)
- Short paragraphs (2-4 sentences max)
- Use bullet points and numbered lists to break up dense information
- Include concrete examples — don't just state principles, show them
- Write at an 8th-grade reading level — clear, not dumbed down
- Active voice. Direct address ("you"). Present tense where possible.

**SEO:**
- Include the target keyword in: title, first 100 words, at least 2 H2s, and the conclusion
- Use related keywords naturally throughout (don't stuff)
- Write a meta description (under 160 characters) that includes the keyword and compels a click
- Suggest internal links to other content on the site if relevant
- Aim for 1,200–2,000 words unless the user specifies otherwise

**Engagement:**
- Break up the text with formatting: bold key phrases, use blockquotes for emphasis
- Include at least one "tweetable" takeaway — a single sentence someone would want to share
- End sections with a transition to the next (but keep them natural, not formulaic)

### 4. Polish

After drafting:
1. Re-read for brand voice consistency — does this sound like the personality?
2. Cut fluff — if a sentence doesn't add value, remove it
3. Check that every section delivers on its heading's promise
4. Verify the CTA is clear and aligned with the brand's goals
5. Add the meta description and suggested slug

### 5. Deliver

Present the final post in this format:

```
## Meta
- **Title:** [Post title]
- **Slug:** /blog/[suggested-url-slug]
- **Meta Description:** [Under 160 chars]
- **Target Keyword:** [Primary keyword]
- **Word Count:** [Approximate]

---

[Full blog post content in markdown]
```

## Quality Checklist

Before delivering, verify:
- [ ] Opens with a hook, not a cliché
- [ ] Every section has a clear point
- [ ] Brand voice is consistent throughout
- [ ] Target keyword appears in title, intro, H2s, and conclusion
- [ ] Paragraphs are short (2-4 sentences)
- [ ] Includes concrete examples or data
- [ ] Has a clear call to action
- [ ] Meta description is compelling and under 160 characters
- [ ] Word count is 1,200–2,000 (unless specified otherwise)
